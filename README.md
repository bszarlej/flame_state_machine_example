# flutter_state_machine_example

## Running the Example

### Prerequisites

Make sure you have Flutter installed:

```bash
flutter --version
```

For installation instructions, see the official Flutter documentation:
https://docs.flutter.dev/get-started/install

### Clone the repository

```bash
git clone https://github.com/bszarlej/flame_state_machine_example.git
cd flame_state_machine_example
```

### Install dependencies

```bash
flutter pub get
```

### Run the application

```bash
flutter run
```

### Controls
| Key | Action |
| --- | --- |
| W / A / S / D | Move player |
| Shift | Sprint |
